# Trabajo Final - Módulo 5 (Android con Kotlin)
**Alumno:** Christian Irureta

## Descripción del Proyecto
Este repositorio contiene la lógica de validación para un sistema de **Login y Registro** utilizando arquitectura **MVVM (Model-View-ViewModel)**.

## Implementaciones técnicas:
1. **ViewModel:** Se encarga de la lógica de negocio para validar que los campos no estén vacíos y tengan el formato correcto.
2. **LiveData:** La Activity observa los cambios en el ViewModel para mostrar mensajes al usuario (Toasts).
3. **Kotlin:** Código 100% migrado de Java a Kotlin siguiendo las guías del Módulo 5.
